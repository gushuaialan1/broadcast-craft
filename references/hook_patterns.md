# Hook 钩子模式 / Hook Patterns

> 短视频的第一性原理：**0-3 秒决定生死**。
> 钩子不是"标题的口播版"，钩子是"让用户停止滑动手指的刺激"。

---

## 1. Hook 的底层逻辑

所有 Hook 都在触发人类的某个**原始心理按钮**：

| 按钮 | 心理机制 | 为什么停下 |
|---|---|---|
| **好奇** | 认知闭合需求 | "我知道答案，我就想看你说的和我想的一不一样" |
| **恐惧** | 生存本能 | "这件事可能会影响到我，我必须看完" |
| **愤怒** | 不公平感 | "怎么可以这样？我要看看你怎么骂" |
| **共鸣** | 群体归属 | "这说的就是我啊！" |
| **幻想** | 反差现实 | "原来还可以这样活" |
| **干货** | 效率忌惮 | "我正需要这个，看看他怎么解决" |

**Hook 的黄金法则**：好的 Hook 同时触发 **至少两个按钮**。例如：
- 好奇 + 恐惧 = "你会不会被AI替代？先看看这5个信号" |
- 共鸣 + 愤怒 = "打工人今天又被PUA了，但是这次我不打算忍了" |

## 2. 八种 Hook 原型

### 2.1 反常识型（最安全、最通用）

**结构**："你以为 X 是 Y，但实际上是 Z。"

**例子**：
- "你以为睡够8小时才健康？错。最近研究发现，睡得太多反而会..."
- "大家都说早睡早起身体好，但我一个睡不着的朋友，在看完这个视频后给我发了一条消息..."

**平台适配**：全平台通用
**心理按钮**：好奇 + 干货

---

### 2.2 反问型（强共鸣）

**结构**："你是不是也... ？"

**例子**：
- "你是不是也这样？明明已经很累了，但晚上躺在床上就是睡不着。"
- "有没有人和我一样，每次打开手机想看个视频，一刷就是两个小时..."

**平台适配**：全平台通用，视频号效果最佳
**心理按钮**：共鸣 + 好奇

---

### 2.3 情绪引爆型（最危险、最有效）

**结构**："一个极端情绪的第一人称场景。"

**例子**：
- "今天我被辟了。"【停顿】"就因为我在会议上说了一句真话。"
- "刚才我到底下看到一幕，我站在那里傻了。"

**平台适配**：抖音、快手
**心理按钮**：愤怒/恐惧 + 共鸣
**风险**：情绪必须真实，不然会被评论区反杀

---

### 2.4 数据冲击型（干货专用）

**结构**："一个违反直觉的数据/对比。"

**例子**：
- "我走遍了全国20个城市，发现月薪3000和月薪30000的人，最大的差距不是能力，是这个。"
- "2024年，有100万人做了同一件事。但只有0.1%的人知道真相。"

**平台适配**：B站、视频号
**心理按钮**：干货 + 好奇

---

### 2.5 秘密揭露型（悬念最强）

**结构**："有一件事，很少有人知道..."

**例子**：
- "有一个秘密，所有大厂HR都不会告诉你，但我今天必须说。"
- "我做迗一个很犯规的实验。结果让我夜里睡不着觉。"

**平台适配**：全平台通用
**心理按钮**：好奇 + 恐惧

---

### 2.6 对立冲突型（话题性最强）

**结构**："有两种人，你是哪一种？"

**例子**：
- "有两种人是注定赚不到钱的。今天我来告诉你为什么，以及怎么逃出来。"
- "一种人看到机会就上，另一种人看到机会就怕。今天这条视频，可能会改变你的一生。"

**平台适配**：抖韹、B站
**心理按钮**：共鸣 + 恐惧

---

### 2.7 场景幻化型（视频号、B站特效）

**结构**："如果你...，你会..."

**例子**：
- "如果你可以回到10年前，给当时的自己打一个电话，你会说什么？我先来。"
- "假设明天你赔了一百万，你的生活会变成什么样子？我花了一周时间研究这个问题。"

**平台适配**：视频号、B站
**心理按钮**：幻想 + 共鸣

---

### 2.8 零成本好奇型（转化率最高）

**结构**："不花一分钱，你就能..."

**例子**：
- "不用买课、不用交学费，我把我花了3万块学到的东西，今天免费分享给你。"
- "有一个官方工具是免费的，但所有人都用错了。今天我教你正确用法。"

**平台适配**：全平台通用，快手效果最佳
**心理按钮**：干货 + 幻想

---

## 3. Hook 选择决策树

```
是否有具体数据/对比？
  是 → 数据冲击型
  否 → 是否有强烈个人情绪？
      是 → 情绪引爆型
      否 → 是否能带来“免费收益”？
          是 → 零成本好奇型
          否 → 是否能触发“我也是”？
              是 → 反问型
              否 → 是否有“少数人知道的秘密”？
                  是 → 秘密揭露型
                  否 → 是否可以构建“两种人”对立？
                      是 → 对立冲突型
                      否 → 是否有“反常识”内容？
                          是 → 反常识型
                          否 → 场景幻化型
```

---

## 4. Hook 与平台的配对表

| 平台 | 推荐 Hook | 为什么 |
|---|---|---|
| 抖韹 | 情绪引爆型、反常识型、对立冲突型 | 完播率优先，需要极端刺激 |
| 快手 | 零成本好奇型、反问型、反常识型 | 用户更追求实用性，喜欢"有货" |
| 视频号 | 反问型、场景幻化型、数据冲击型 | 社交属性强，共鸣和转发价值重要 |
| B站 | 数据冲击型、对立冲突型、场景幻化型 | 用户耐心更好，接受干货密度高的开场 |

## 5. Hook 进阶技巧

### 5.1 视觉化 Hook
口播不仅是语言，还是**画面**。好的 Hook 同时触发视觉：
- "你看这个数据"【手指向屏幕】
- "我现在在XXX的现场"【镜头拉远展示环境】
- "这是我刚才录的"【手机屏幕入画】

### 5.2 声音 Hook
用**语气**做钩子，而不是用内容：
- 突然降低音量："有件事...我想了很久..."
- 突然加快语速："我今天必须得告诉你一件事"
- 方言门柜："你们那边是不是也这么说的——"

### 5.3 双 Hook 结构
如果视频超过60秒，建议使用"开头 Hook + 中段第二钩子"：
- 第二钩子在**第三个情绪低谷**出现（通常在20秒左右）
- 第二钩子的作用是"拉住即将滑走的人"
