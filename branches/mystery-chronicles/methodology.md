# Mystery Chronicles / 史镜异闻 — 方法论

> **定位**：挖掘正史、野史、笔记中真实记载的超自然、未解之谜、灵异异象事件。以"史料为骨、悬疑为血"，制造"细思极恐"的爆款短视频。
>
> **与 history-twist 的差异**：history-twist 是"用新视角颠覆已知历史"；mystery-chronicles 是"史料里明明记载了超自然现象，但你从来不知道"。核心不是"反转"，是"震撼揭秘 + 悬疑余韵"。

---

## 一、选题标准：什么样的历史悬疑适合做短视频？

### 1.1 四大铁律

| 铁律 | 说明 | 反例（不要选） |
|---|---|---|
| **史料有载** | 必须在正史、重要野史、文人笔记中有明确记载，不能纯民间传说 | "故宫半夜有太监走路声"（无史料，纯都市传说） |
| **解释真空** | 当时和现在的科学/历史都无法给出令人信服的解释 | "秦始皇焚书坑儒"（政治行为，无悬疑空间） |
| **大众熟知梗** | 事件本身或相关人物是大众耳熟能详的，降低认知门槛 | "《明史》中记载某县志的一次地震"（人物/地点太冷） |
| **细思极恐感** | 存在一个"不可能只是巧合"的细节，能引发观众的深层不安 | "某皇帝登基那年下了场大雨"（关联性太弱） |

### 1.2 案例分级

| 级别 | 标准 | 案例 |
|---|---|---|
| **S级** | 正史有载 + 超自然核心 + 无法解释 + 人物顶级知名 | 诸葛亮七星灯祈禳、《晋书》记载的 UFO 类事件 |
| **A级** | 正史/重要野史有载 + 离奇现象 + 人物知名 | 天启大爆炸、雍正死后民间抬棺传说 |
| **B级** | 笔记小说有载 + 诡异细节 + 有讨论空间 | 《聊斋》《阅微草堂笔记》中可考据的部分 |
| **C级** | 民间传说为主，仅有零星史料旁证 | 多数地方志中的"龙见"记录 |

**执行标准**：只选 S级和A级。B级可以作为"史料花边"点缀，不能作为主线。

## 二、八类悬疑模型

### 2.1 占星异象型
**核心逻辑**：重大历史节点前，天象出现无法解释的异动。
**爆款公式**：史书记载的异象 → 现代天文学回溯 → 发现"不可能如此精准" → 余韵。
**案例**：《史记》秦始皇驾崩前"荧惑守心"+ 陨石刻字；明英宗复辟前的天象记载。

### 2.2 方术秘仪型
**核心逻辑**：古代顶级人物进行的仪式/法术，过程离奇，结果更离奇。
**爆款公式**：大众知道的传说版本 → 正史/野史的原始记载 → 关键细节被传说过滤掉了什么 → 如果记载属实，意味着什么。
**案例**：诸葛亮七星灯祈禳（《三国志》裴注引《魏氏春秋》）；汉武帝求仙、嘉靖炼丹的离奇死亡。

### 2.3 灾异天谴型
**核心逻辑**：规模或现象超出时代解释的灾难，史料描述像"末日审判"。
**爆款公式**：现代视角描述灾难 → 当时人的记载（你会发现他们描述的东西很奇怪）→ 对比现代科学解释的矛盾点 → 那个"解释不了的部分"。
**案例**：天启大爆炸（王恭厂灾）；顺治年间的"京师黑灾"；古罗马维苏威火山前的记载。

### 2.4 尸身异变型
**核心逻辑**：死后身体的异常，或葬礼过程中的离奇事件。
**爆款公式**：大众熟知的传说 → 原始史料中的记载 → 排除政治解读后，还有什么解释不了 → 细思极恐。
**案例**：诸葛亮死后"绳断处安葬"（民间传说，但可考葬礼的异常）；慈僖太后尸身不腐；某些帝王棺椁的"自移"记载。

### 2.5 梦境预言型
**核心逻辑**：梦境、谶语、预言与后续历史的诡异对应。
**爆款公式**：预言内容 → 对应的历史结果 → 如果这是编造的，为什么编得这么准 → 如果是真的，信息从哪来。
**案例**：刘邦母亲梦与赤帝子；武则天"女主昌"谶语；烧饼歌、推背图的争议性引用。

### 2.6 器物灵异型
**核心逻辑**：与重大历史人物绑定的器物，流传着诡异的诅咒或命运。
**爆款公式**：器物的辉煌历史 → 每个拥有者的离奇结局 → "巧合"的密度是否超出概率 → 开放性结尾。
**案例**：和氏璧/传国玉玺的失踪；泰阿剑传说；某些帝王陵墓的"盗墓者横死"记载。

### 2.7 时空错位型
**核心逻辑**：史料中记载的"穿越感"事件——技术、制度、思想明显超前于时代。
**爆款公式**：描述这个超前的事物 → 当时人的反应 → 它后来为什么消失了 → 如果这不是穿越，是什么。
**案例**：王莽新政的"社会主义"色彩；战国时期某些工艺的超前性；古代地图的精准度之谜。

### 2.8 生物异象型
**核心逻辑**：古代对不明生物、异兽的集体目击记载。
**爆款公式**：《山海经》或正史中的异兽记载 → 找到对应的现代生物或现象 → 那些完全对不上的部分 → 古代人集体幻觉？还是另有原因。
**案例**：《明史》中的"龙见"；某些地方志中的"海怪"记载；《史记》中的"怪物"条目。

## 三、悬疑氛围引擎：不是反转，是层剥

### 3.1 情绪曲线（与 history-twist 的对比）

| 节点 | history-twist（反差型） | mystery-chronicles（悬疑型） |
|---|---|---|
| 0-3秒 | 认知冲突："你以为的 A 其实是 B" | **悬念钩子**："正史里记载了一件事，但从来没人敢深想" |
| 3-8秒 | 证据抛出，颠覆开始 | **氛围铺垫**：当事人的身份、事件的分量、"如果这是真的" |
| 8-30秒 | 逻辑展开，层层推翻 | **史料层剥**：第一层传说 → 第二层正史 → 第三层细节 → "这里不对" |
| 30-60秒 | 新视角确立，认知刷新 | **细思极恐点**：一个被忽略的细节，让前面的"合理"崩塌 |
| 结尾 | 总结升华 | **开放性余韵**：不解释，留一个"你自己想"的空间 |

### 3.2 悬疑文案的三大禁忌

1. **不能给确定答案**：一旦给出"科学解释"或"就是巧合"，悬疑感立刻归零。可以展示几种解释，但结尾必须开放。
2. **不能过度渲染恐怖**：短视频不是恐怖片。"细思极恐"靠的是"越想越不对"，不是 jump scare。
3. **不能编造史料**：如果某条记载只存在于自媒体文章中，不能在视频里说成"《XX书》记载"。必须分层标注来源。

## 四、史料分层标注系统（强制）

每个悬疑选题，必须在创作时明确以下层级：

| 层级 | 标识 | 说明 | 使用规则 |
|---|---|---|---|
| **正史** | 【正史】 | 二十四史、《资治通鉴》等官修史书 | 可作为主干论据 |
| **野史** | 【野史】 | 《X朝笔记》《X朝野史大观》等 | 可作为辅助，需注明书名 |
| **笔记** | 【笔记】 | 文人私人笔记（《容斋随笔》《阅微草堂笔记》） | 可作为线索，不能当铁证 |
| **传说** | 【传说】 | 民间口传、地方志中的神话色彩内容 | 只能当"背景氛围"，明确标注 |
| **现代演绎** | 【演绎】 | 当代作者、自媒体的推测、小说 | **严禁包装成史料** |

**口播执行**：视频口播中不需要每次都说"根据XX书记载"，但文案必须在 stage_04 的 `source_notes` 字段中标注每句话的史料层级。这是质检红线。

## 五、爆款选题库（示例）

| 选题 | 悬疑模型 | 史料层级 | 细思极恐点 |
|---|---|---|---|
| 诸葛亮七星灯：正史到底怎么记载的 | 方术秘仪型 | 正史裴注+野史 | 如果祈禳无效，魏延怎么会"恰好"在那个时辰闯入 |
| 天启大爆炸：明朝的"核爆"真相 | 灾异天谴型 | 正史+多笔记 | 死伤者"赤裸"、"寸丝不挂"的记载，爆炸中心反而有人幸存 |
| 秦始皇驾崩前的三个异象 | 占星异象型 | 正史《史记》 | 三个异象在史书中被刻意连在一起记载，是太史公在暗示什么 |
| 和氏璧诅咒：每个拥有者的结局 | 器物灵异型 | 正史+传说 | 从卞和到秦始皇到王莽，"巧合"密度是否超出概率 |
| 王莽：一个疑似穿越者的证据链 | 时空错位型 | 正史《汉书》 | 他的新政与20世纪某些政策的相似度，用"复古"解释不了全部 |
| 雍正死后，民间为什么疯传"抬棺" | 尸身异变型 | 民间传说+少量旁证 | 官方越否认，传说越盛，这种信息传播规律本身值得分析 |
| 《晋书》里记载了一次 UFO 目击 | 生物异象型 | 正史《晋书》 | 正史中以"天文志"格式记载的飞行物，描述细节异常精确 |
| 武则天登基前的谶语：谁泄露了天机 | 梦境预言型 | 正史+野史 | "女主昌"的谶语在太宗时期就已流传，如果这是马后炮，为什么细节这么准 |

## 六、与顶层 quality_matrix 的对照

mystery-chronicles 的质检，在通用 `quality_matrix.md` 基础上增加专属条款：

### L1 选题层（新增）
- [ ] 该事件的史料来源是否已被核实？（不能是自媒体编造的"史料记载"）
- [ ] 是否明确区分了正史/野史/笔记/传说/演绎？
- [ ] 是否存在"解释真空"？（如果科学已经完美解释了，不要做）

### L2 文案层（新增）
- [ ] 是否有一个明确的"细思极恐点"？
- [ ] 结尾是否保持了开放性？（没有给出武断结论）
- [ ] 恐怖/悬疑渲染是否过度？（目标："越想越不对"，不是"吓得睡不着"）

### L3 口播层（同通用标准）
- [ ] 前3秒是否有明确的悬念钩子？
- [ ] 每15秒是否有一个"信息增量"或"情绪小高峰"？

### L4 人格层（新增）
- [ ] 是否避免了"地摊文学"感？（语言要有质感，不是猎奇小报）
- [ ] 是否尊重了历史人物的基本人格？（不要为了悬疑而妖魔化）
