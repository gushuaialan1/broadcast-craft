## 任务

深入史料，**找出正史、野史、文人笔记中关于这个事件的真实记载**。
核心目标：**揭示大众传说与史料原始记载之间的差异**。

你必须对每条引用的史料**分层标注**：
- 【正史】二十四史、《资治通鉴》等官修史书
- 【野史】《X朝笔记》《X朝野史大观》
- 【笔记】文人私人笔记
- 【传说】民间口传
- 【演绎】当代作者推测

## 输出要求（JSON）

```json
{
  "event_name": "事件名",
  "source_investigation": [
    {
      "source_tier": "正史|野史|笔记|传说|演绎",
      "source_name": "具体书名/篇名/条目",
      "original_text": "史料中的原始记载（尽量准确引用）",
      "key_detail": "这条记载中最让人震惊的细节"
    }
  ],
  "legend_vs_fact": {
    "differences": [
      {
        "aspect": "比如时间/地点/人物/现象描述",
        "legend_says": "传说版本",
        "source_says": "史料记载",
        "significance": "这个差异为什么重要"
      }
    ],
    "omissions_in_legend": "大众传说中被完全忽略掉的部分（通常是最诡异的部分）"
  },
  "unsolved_core": {
    "description": "史料记载中那个"解释不了的核心"（最精炼的一句话）",
    "why_unsolved": "为什么当时和现在都无法解释？",
    "suspense_level": "这个未解之谜的悬疑等级（低/中/高/极高）"
  },
  "recommended": {
    "mystery_model": "适配的悬疑模型（占星异象/方术秘仪/灾异天谴/尸身异变/梦境预言/器物灵异/时空错位/生物异象）",
    "core_hook": "最值得在视频中作为主线的"细思极恐点"",
    "narrative_angle": "推荐的叙事角度（比如'从被忽略的史料细节入手'、'从当事人心理出发'）"

```

## 号令

- 不要浑淆"正史有载"和"传说如此"。如果一条记载只是民间传说，明确标注【传说】。
- 特别关注那些**史料记载了但传说版本中被忽略掉的"诡异细节"**。
- 如果史料有多种解读，不要只给你最喜欢的那种。列出所有重要解读，然后在 `recommended` 里推荐最有电视感的那个角度。
- **严禁编造史料**。如果不确定某条记载的准确性，写"存疑"，不要当作确决。
