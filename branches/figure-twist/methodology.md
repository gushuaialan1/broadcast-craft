# Figure Twist Methodology – 人物揭秘型短视频文案方法论

## 1. 核心定位

不是讲"历史事件的新视角"，而是讲**"这个人不是你想的那样"**。

观众爽感来源：**"人设崩塌"或"人设补完"**——你原以为他是 A，其实他是 B/C/D。

## 2. 8 种人物揭秘模型

| 模型 | 核心操作 | 经典选题 |
|:---|:---|:---|
| **人设崩塌型** | 剥掉大众标签，暴露真实动机与利益计算 | 刘备的"仁德"是表演还是生存策略？ |
| **侧面补完型** | 挖掘被史书/影视忽略的 A 面 | 曹操不只是奸雄，还是建安文学领袖、失败的父亲 |
| **标签误读型** | 指出标签本身的逻辑漏洞或时代错位 | 海瑞是清官，但他真的懂治国吗？ |
| **情感反转型** | 强悍外表下的脆弱、恐惧与代价 | 李世民杀兄后的终身噩梦与自我惩罚 |
| **能力盲区型** | 纠正被高估或低估的具体能力 | 诸葛亮的军事才能被《三国演义》高估了吗？ |
| **时代牺牲品型** | 人物被时代结构绑架，非个人品行问题 | 崇祯不是昏君，是王朝积弊的替罪羊 |
| **小人物高光型** | 改变大人物命运的关键小人物 | 给项羽指路的田父，一个农夫毁了楚霸王 |
| **关系重构型** | 人物关系的权力真相，超越道德叙事 | 和珅与乾隆：不只是贪腐，更是权力代理人 |

## 3. 五阶工作流

```
标签锚定 → 深度挖掘 → 钩子设计 → 人格拼贴 → 爆点质检
```

| 阶段 | 名称 | 核心任务 |
|:---|:---|:---|
| Stage 01 | **Tag Anchor** | 锁定大众标签，分析标签来源与情绪基调 |
| Stage 02 | **Depth Mining** | 从史料中找反标签证据，选定揭秘模型 |
| Stage 03 | **Hook Design** | 设计"标签 vs 真相"的反差钩子 |
| Stage 04 | **Compose** | 300-450 字口播稿，人格拼贴式叙事 |
| Stage 05 | **Review** | 五问质检：反差够强？史料可靠？人设立体？情绪收获？值得分享？

## 4. 人格拼贴叙事结构

```
[开场钩子]     一句话颠覆认知（3秒）
[标签还原]     大众怎么看他？先共情观众（10秒）
[证据插入]     丢一个反标签的硬核史料（15秒）
[动机重构]     他为什么这么做？时代/利益/性格（30秒）
[第二证据]     再补一个侧面，让人物立起来（20秒）
[人格拼图]     总结：他不是 A，他是 A+B+C（20秒）
[现代映照]     这个人物侧面在今天意味着什么？（15秒）
[结尾余韵]     开放性金句，引发评论区争论（10秒）
```

## 5. 史料使用原则

- **正史优先**：陈寿《三国志》、范晔《后汉书》、司马光《资治通鉴》可直接引用
- **野史辅助**：《世说新语》《容斋随笔》可作侧面，需标注来源
- **严禁演义**：《三国演义》《隋唐演义》的桥段必须区分，不能当史料
- **影视区分**：94 版三国、军师联盟等影视形象与历史形象的差异，本身就是选题

## 6. 安全红线

1. **不洗白恶行**：可以解释动机，不能为屠杀、暴政开脱
2. **不污名化英雄**：可以指出盲区，不能全盘否定正面贡献
3. **性别与民族敏感**：涉及女性人物、少数民族政权时慎用"权谋""背叛"叙事
4. **现代政治映射**：古代人物分析不得影射当代政治人物

## 7. 爆点选题库（示例）

| 人物 | 大众标签 | 揭秘方向 | 模型 |
|:---|:---|:---|:---|
| 刘备 | 仁德之君 | 他的"哭"是真情还是政治表演？ | 人设崩塌型 |
| 曹操 | 奸雄 | 他写给妻子的遗书，暴露了什么？ | 侧面补完型 |
| 海瑞 | 清官 | 他亲手饿死五岁女儿，真的吗？ | 标签误读型 |
| 李世民 | 明君 | 玄武门之后，他睡得好吗？ | 情感反转型 |
| 诸葛亮 | 智圣 | 六出祁山，军事上真的明智吗？ | 能力盲区型 |
| 崇祯 | 昏君 | 勤政十七年，为何还是亡了国？ | 时代牺牲品型 |
| 田父 | 无 | 一个指错路的农夫，葬送了项羽 | 小人物高光型 |
| 和珅 | 贪官 | 乾隆为什么离不开他？ | 关系重构型 |
