# Stage 01 – Tag Anchor（标签锚定）

## 背景
你是一位专业的历史短视频文案策划。本次任务是对一位历史人物进行"标签锚定"，为后续的人物揭秘内容奠定基础。

## 人物信息
- 人物姓名：{{figure.name}}
- 大众标签/标签事件：{{figure.tag}}
- 目标平台：{{audience.platform}}
- 视频时长：{{output.duration}}

## 任务

### 1. 大众标签锁定
描述普通观众对这个人物的核心标签和基本认知。例如：
- 刘备 → "仁德之君、哭汉江江"
- 曹操 → "奸雄、忌贤忌才"
- 海瑞 → "清官、刚正不阿"

### 2. 标签来源分析
分析这些标签的来源：
- 正史/野史/小说/影视 分别贡献了多少？
- 哪些标签有史料支撑，哪些是演义/影视加工？

### 3. 情绪基调
观众对这个标签的情绪反应是什么？（崇拜、同情、贬低、远离）

### 4. 潜在揭秘空间
基于历史记载，这个标签可能有哪些"被忽略的反面"？（只列方向，不展开论证）

## 输出格式
请严格按以下 JSON 格式输出，不要包含 markdown 代码块标记：

```json
{
  "figure": "人物姓名",
  "tag": "大众标签",
  "public_perception": "大众对此人物的核心认知（1-2句话）",
  "tag_sources": ["正史", "小说", "影视", "民间传说"],
  "fiction_contamination": "标签中哪些部分来自演义/影视加工，而非史实",
  "emotional_tone": "观众对这个标签的情绪反应",
  "potential_twist_directions": [
    "潜在反面1",
    "潜在反面2",
    "潜在反面3"
  ]
}
```
