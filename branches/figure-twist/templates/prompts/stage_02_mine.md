# Stage 02 – Depth Mining（深度挖掘）

## 背景
基于 Stage 01 的标签锚定，现在需要从史料中挖掘反标签证据，选定最有爆点潜力的揭秘角度。

## 前置状态
```json
{{state.stage_01 | tojson(indent=2)}}
```

## 任务

### 1. 史料搜索（模拟）
列出 3-5 条可以支撑"反标签"的史料线索。包括：
- 正史记载（优先）
- 野史笔记
- 人物的个人诗文、书信、行为记载

### 2. 揭秘角度定位
从以下 8 种模型中选择最合适的 1-2 种：
- 人设崩塌型 / 侧面补完型 / 标签误读型 / 情感反转型
- 能力盲区型 / 时代牺牲品型 / 小人物高光型 / 关系重构型

### 3. 史料可信度评级
对每条证据进行可信度评级：
- A级：正史明确记载
- B级：野史/笔记有具体出处
- C级：传说/推测，需谨慎使用

### 4. 推荐策略
提供最推荐的揭秘方向、核忄反差点、证据链路。

## 输出格式
请严格按以下 JSON 格式输出：

```json
  }
}
```
